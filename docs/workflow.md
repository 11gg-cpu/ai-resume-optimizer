# Workflow

## 1. Job Description Analysis

Break the JD into role goals, hard skills, soft skills, domain keywords, and evidence requirements.

## 2. Candidate Evidence Mapping

Map each requirement to a project, coursework, tool, or measurable experience.

## 3. Keyword and Gap Review

Check whether the resume naturally includes the most important role terms. Missing terms are only added when they reflect real experience.

## 4. Bullet Rewriting

Rewrite bullets using:

- Action
- Method
- Tool
- Outcome
- Role relevance

## 5. Human Verification

Remove inflated claims, unsupported metrics, and language that sounds too generic.
